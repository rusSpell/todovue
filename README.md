# todolist

## О проекте
```
- Веб-сервис - список задач
- SPA на Vue.js (vue-cli, vuex, vue router)
- Bulma CSS framework
- Axios для работы с API
- Реализована авторизация
- Создание задач (локально и на сервере по API)
- Редактирование задач
```
## Project setup
```
npm install
```
